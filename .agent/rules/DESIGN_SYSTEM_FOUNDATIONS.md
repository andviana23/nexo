O NEXO segue o Design System oficial.

Obrigatório:
- Usar tokens globais definidos em CSS Variables: --primary, --foreground, --destructive, etc.
- Usar Tailwind como motor principal de styling.
- Tipografia: Inter (padrão) e JetBrains Mono (código/tabelas).
- Acessibilidade é obrigatória: foco, aria-labels, contraste, navegação por teclado.
- Nunca criar cor manual (#HEX). Sempre usar tokens.
- Nunca usar margin/padding aleatórios; sempre usar escala do DS.

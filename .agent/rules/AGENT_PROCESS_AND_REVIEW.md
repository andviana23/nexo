Antes de sugerir qualquer código, o Agent deve:

1. Ler PRD e fluxos do módulo afetado.
2. Ler documentos de arquitetura.
3. Ler o Design System.
4. Verificar RBAC.
5. Verificar multi-tenant.
6. Validar se DTO, models e use cases existem.
7. Apontar inconsistências no sistema antes de implementar.

O Agent deve recusar soluções que quebrem arquitetura, design system ou multi-tenant.
